# Code Review Flow

This project is a lightweight CrewAI-style code review flow scaffold created from the `C1M3_Assignment.ipynb` assignment structure.

It provides a clean starting layout for organizing agents, tasks, tools, guardrails, and utility files for an automated code review workflow.

## Project structure

- `C1M3_Assignment.ipynb` - original assignment notebook for this project
- `pyproject.toml` - project configuration file
- `src/code_review_flow/main.py` - main entry point placeholder
- `src/code_review_flow/utils.py` - shared utility placeholder
- `src/code_review_flow/crews/code_review_crew/` - crew package for the code review workflow
- `src/code_review_flow/crews/code_review_crew/config/agents.yaml` - agent configuration placeholder
- `src/code_review_flow/crews/code_review_crew/config/tasks.yaml` - task configuration placeholder
- `src/code_review_flow/crews/code_review_crew/guardrails.py` - guardrails placeholder
- `src/code_review_flow/tools/custom_tool.py` - custom tool placeholder
